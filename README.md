# 📚 Advanced Vocabulary Challenge

This is an interactive mini web app designed to help learners practice advanced English vocabulary in context. It features a fill-in-the-blank exercise with tooltips and animated background, styled with a modern and responsive design.

## ✨ Features

- 🧠 Practice 5 advanced English adjectives in real-world contexts
- 💬 Tooltips with definitions (hover to reveal)
- ✅ Instant feedback on correct and incorrect answers
- ⚠️ Animated result alert with color-coded feedback
- 🎨 Responsive layout with gradient background

## 🛠️ Technologies Used

- HTML5
- CSS3 (including animations & responsive layout)
- JavaScript (vanilla)

## 📦 How to Use

1. Clone the repository or download the ZIP.
2. Open `index.html` in your browser.
3. Fill in the blanks using appropriate vocabulary.
4. Click **"Check Answers"** to see how you did.
5. Hover over the ❓ to get a definition hint.
6. Click anywhere or the ✖ button to close the result alert.
